# Pacman

A Pacman game clone - not that faithful to the original one - made with C++ and SDL2.

## Running
### Build and run the standalone target

Use the pre-made script to build, then run the standalone target:

```bash
cd Pacman
./scripts/build.sh
./build/bin/PacMan
```

### Running tests

Use the pre-made script to run tests:

```bash
cd Pacman
./scripts/test.sh
./scripts/style.sh
```
